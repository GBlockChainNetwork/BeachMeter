# Contributing to BeachMeter

Mulțumim pentru interes!

1. Fork repo
2. Create branch (`feature/nume-feature`)
3. Commit changes (`git commit -m 'Add some feature'`)
4. Push to the branch (`git push origin feature/nume-feature`)
5. Open a Pull Request.

Vezi [TECHNICAL_PLAN.md](docs/TECHNICAL_PLAN.md) pentru convenții.
