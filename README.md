# BeachMeter

**Aplicație mobilă pentru monitorizarea UV și condițiilor de plajă – Protejează-te inteligent la soare!** 🌊☀️

BeachMeter te ajută să te bucuri de plajă în siguranță cu alerte UV recurente, prognoze meteo hyper-locale și recomandări personalizate.

## Descriere
Vezi [docs/TECHNICAL_PLAN.md](docs/TECHNICAL_PLAN.md) pentru detalii complete.

## Features
- UV Index live + forecast (NASA/NOAA/Open-Meteo)
- Alerte inteligente bazate pe GPS și expunere
- Hartă plaje + condiții reale (valuri, apă, salvamari)
- Personalizare pe tip piele (Fitzpatrick)
- Premium: tracking sesiuni, AI insights
